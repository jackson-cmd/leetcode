https://leetcode.com/problems/number-of-1-bits/

time complexity: O(1)
space complexity: O(1)
```
public class Solution {
    // you need to treat n as an unsigned value
    int ans = 0;
    public int hammingWeight(int n) {
        while(n != 0) {
            ans++;
            n &= n-1;
        }
        return ans;
    }
}
```
