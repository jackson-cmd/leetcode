time complexity: O(n)
space complexity: O(1)

faster way
```
class Solution {
    public int lengthOfLastWord(String s) {
        // trim the trailing spaces
        int p = s.length() - 1;
        while (p >= 0 && s.charAt(p) == ' ') {
            p--;
        }

        // compute the length of last word
        int length = 0;
        while (p >= 0 && s.charAt(p) != ' ') {
            p--;
            length++;
        }
        return length;
    }
}
```
built-in function
```
class Solution {
    public int lengthOfLastWord(String s) {
        String[] tmp = s.trim().split("\s+");
        return tmp[tmp.length - 1].length();
    }
}
```
