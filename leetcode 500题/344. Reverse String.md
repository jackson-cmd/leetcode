https://leetcode.com/problems/reverse-string/

time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public void reverseString(char[] s) {
        int l = 0, r = s.length - 1;
        while (l < r) {
            char tmp = s[l];
            s[l++] = s[r];
            s[r--] = tmp;
        }
    }
}
```
