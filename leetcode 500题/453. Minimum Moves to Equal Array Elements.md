https://leetcode.com/problems/minimum-moves-to-equal-array-elements/

time complexity: O(nlogn)
space complexity: O(1)
```
class Solution {
    public int minMoves(int[] nums) {
        int l = 0, r = nums.length - 1, ans = 0;
        Arrays.sort(nums);
        while(r > 0) {
            ans += nums[r--] - nums[0];
        }
        return ans;
    }
}
```
