https://leetcode.com/problems/remove-duplicates-from-sorted-array/

time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public int removeDuplicates(int[] nums) {
        int n = nums.length;
        if(n == 0) return n;
        int i = 0;
        for(int j = 0; j < n; ++j) {
            if(nums[j] != nums[i]) {
                i++;
                nums[i] = nums[j];
            }
        }
        return i + 1;
    }
}
```
