https://leetcode.com/problems/excel-sheet-column-title/

time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public String convertToTitle(int c) {
        StringBuilder sb = new StringBuilder();
        while(c > 0) {
            c -= 1;
            int last = c%26;
            sb.append((char)(last + 'A'));
            c /= 26;
        }
        return sb.reverse().toString();
    }
}
```
