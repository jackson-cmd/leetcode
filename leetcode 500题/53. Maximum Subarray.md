https://leetcode.com/problems/maximum-subarray/


time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public int maxSubArray(int[] nums) {
        int ans = nums[0], cur = nums[0];
        for(int i = 1; i < nums.length; ++i) {
            cur = Math.max(nums[i], cur + nums[i]);
            ans = Math.max(ans, cur);
        }
        return ans;
    }
}
```
