time complexity: O(n)
space complexity: O(n)
```
class Solution {
    public ListNode swapPairs(ListNode head) {
        if(head == null || head.next == null) return head;
        ListNode a = head, b = head.next;
        ListNode tmp = swapPairs(b.next);
        b.next = a;
        a.next = tmp;
        return b;
    }
}
```
