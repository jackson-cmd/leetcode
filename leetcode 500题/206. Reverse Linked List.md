time complexity: O(n)
space complexity: O(n)
```
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null || head.next == null) return head;
        ListNode next = head.next;
        ListNode ans = reverseList(head.next);
        next.next = head;
        head.next = null;
        return ans;
    }
}
```
