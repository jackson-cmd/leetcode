https://leetcode.com/problems/merge-sorted-array/

time complexity: O(m+n)
space complexity: O(1)

iterate from end to start，不难
```
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        //from right to left
        int l = m + n - 1;
        int i1 = m - 1, i2 = n - 1;
        while(i2 >= 0 && l >= 0) {
            if(i1 >= 0) {
                if(nums1[i1] > nums2[i2]) {
                nums1[l] = nums1[i1];
                i1--;
                l--;
            } else {
                nums1[l] = nums2[i2];
                i2--;
                l--;
            }
            }else {
                nums1[l] = nums2[i2];
                i2--;
                l--;
            }
        }
    }
}
```
