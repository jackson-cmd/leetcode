https://leetcode.com/problems/best-time-to-buy-and-sell-stock/

time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public int maxProfit(int[] p) {
        int n = p.length;
        int ans = 0;
        int tmin = p[0];
        for(int i = 1; i < n; ++i) {
            tmin = Math.min(tmin, p[i]);
            ans = Math.max(p[i] - tmin, ans);
        }
        return ans;
    }
}
```
