time complexity: O(n)
space complexity: O(1)
```
class Solution {
    public int removeDuplicates(int[] nums) {
        int i = 1;
        int count = 1;
        for(int j = 1; j < nums.length; ++j) {
            if(nums[j] == nums[j-1]) {
                count++;
            } else count = 1;
            if(count <= 2) nums[i++] = nums[j];
        }
        return i;
    }
}
```
